TECHNICAL DOSSIER

Tổng quan dự án

Hầu hết các công cụ chạy ứng dụng Windows trên Linux hiện nay đều tạo ra sự xáo trộn hoặc yêu cầu cấu hình thủ công phức tạp. WinNest giải quyết vấn đề này bằng cách tiếp cận hướng đối tượng và tự động hóa toàn bộ vòng đời ứng dụng.

Mỗi ứng dụng được cài đặt qua WinNest sẽ được cấp một vùng chứa prefix Wine cô lập duy nhất. WinNest tự động quản lý tệp cấu hình ứng dụng (`app.json`), ghi nhận log hoạt động, sao lưu trình cài đặt gốc và tích hợp phím tắt khởi chạy trực tiếp lên thanh menu của Linux Desktop.

winnest-shell --demo

SPECIFICATION

Các đặc tính kỹ thuật

Cô lập Wine Prefix tối đa

Mỗi ứng dụng có registry và cấu hình thư viện DLL riêng biệt, triệt tiêu hoàn toàn khả năng xung đột thư viện giữa các phần mềm khác nhau.

Spawning an toàn cấp thấp

Loại bỏ hoàn toàn việc thực thi chuỗi lệnh shell thô (shell execution string). Thay thế bằng việc khởi tạo tiến trình dạng mảng đối số độc lập để chống khai thác injection.

MIME & Desktop Integration

Đăng ký định dạng MIME tích hợp chặt chẽ với các chương trình quản lý tệp tin Linux (Nautilus, Dolphin). Hỗ trợ nhấp đúp tệp tin để mở bằng phần mềm Windows tương ứng.

Lọc log & Cấu hình tập trung

Tự động bắt luồng xuất (stdout/stderr) của ứng dụng Windows, ghi ra tệp nhật ký JSON có cấu trúc rõ ràng tại thư mục gốc của app.

ARCHITECTURE BLUEPRINT

Sơ đồ thiết kế hệ thống

Kiến trúc WinNest được xây dựng hướng tới sự tách biệt hoàn toàn giữa nhân Core CLI điều hành trực tiếp Wine và giao diện người dùng Desktop (Electron GUI).

Electron GUI (Frontend) Desktop Window

Giao diện quản lý thân thiện, gọi các tiến trình thông qua cơ chế IPC an toàn. Không bao giờ trực tiếp gọi Wine runner.

▼ IPC (Inter-Process Communication)
WinNest CLI / Core Engine TypeScript Core

Tập hợp các bộ điều phối: Quản lý prefix, Parser metadata (`app.json`), Đăng ký MIME desktop launchers và lock tiến trình.

▼ Safe Spawn (Argument Arrays)
Wine Runner Environment Wine Host

Tiến trình Wine và Wineserver cô lập, thực thi mã ứng dụng Windows trực tiếp trên kernel Linux với cấu hình sandbox.
